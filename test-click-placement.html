🎯 Click & Place 기능 테스트

테스트 순서:

1단계: 애플리케이션 열기
2단계: 좌측 패널에서 가구 썸네일 클릭
- 썸네일을 한 번 클릭하면 선택됨 (하이라이트 표시)
- 콘솔에 ✅ 썸네일 선택: [module_id] 로그 확인
3단계: 3D 뷰어에서 슬롯에 마우스 호버
- 마우스를 슬롯 위로 이동하면 고스트 프리뷰가 나타남
- 콘솔에 🖱️ 마우스 이동 - 슬롯 인덱스: [number] 로그 확인
4단계: 원하는 슬롯에서 클릭
- 고스트가 표시된 슬롯을 클릭하면 가구가 배치됨
- 콘솔에 ✅ 가구 "[name]"을 슬롯 [number]에 배치했습니다. 로그 확인

콘솔 로그 확인:

브라우저 개발자 도구(F12)를 열어 Console 탭에서 다음 로그들을 확인하세요:

  • 🎯 [ModuleGallery] Click & Place toggled - 썸네일 클릭 시
  • 🎯 SlotDropZones - selectedModuleForPlacement - 선택 상태 변경 시
  • 🖱️ 마우스 이동 - 슬롯 인덱스 - 마우스 호버 시
  • 🖱️ 클릭 이벤트 - 슬롯 클릭 시
  • 👻 고스트 프리뷰 체크 - 고스트 렌더링 시

예상 동작:

  1. 썸네일 클릭 → 썸네일이 선택 상태로 변경
  2. 3D 뷰어의 슬롯에 마우스 호버 → 반투명 고스트 프리뷰 표시
  3. 슬롯 클릭 → 가구 배치 완료
  4. 썸네일 다시 클릭 → 선택 해제
