## 7. 데이터베이스 스키마

### 7.1 관리자 권한
```
collection: admins
- userId: string
- role: 'super' | 'admin' | 'moderator'
- permissions: string[]
- createdAt: timestamp
- updatedAt: timestamp
```

### 7.2 감사 로그
```
collection: auditLogs
- adminId: string
- action: string
- targetType: 'user' | 'project' | 'system'
- targetId: string
- details: object
- timestamp: timestamp
- ipAddress: string
```

## 8. API 엔드포인트

### 8.1 사용자 관리
- `GET /admin/users` - 사용자 목록 조회
- `GET /admin/users/:id` - 사용자 상세 조회
- `PATCH /admin/users/:id` - 사용자 정보 수정
- `DELETE /admin/users/:id` - 사용자 삭제

### 8.2 프로젝트 관리
- `GET /admin/projects` - 프로젝트 목록 조회
- `GET /admin/projects/:id` - 프로젝트 상세 조회
- `DELETE /admin/projects/:id` - 프로젝트 삭제

### 8.3 통계
- `GET /admin/stats/users` - 사용자 통계
- `GET /admin/stats/projects` - 프로젝트 통계
- `GET /admin/stats/dashboard` - 대시보드 통계
