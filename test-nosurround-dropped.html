노서라운드 + 단내림 가구 배치 테스트

🔧 수정 사항

  • ColumnIndexer.ts: 빌트인 노서라운드 모드에서 이격거리 무시 처리
  • ColumnIndexer.ts: 단내림 없는 경우 빌트인 actualInternalWidth 계산 추가
  • ColumnIndexer.ts: 단내림 있는 경우 빌트인 처리 로직 추가 (왼쪽/오른쪽 모두)
  • SpaceCalculator.ts: 빌트인 노서라운드에서 이격거리 무시

📋 테스트 케이스

테스트 1: 노서라운드 + 빌트인 + 단내림(오른쪽)

1. 새 디자인 생성
2. 서라운드 타입: 노서라운드 선택
3. 설치 타입: 빌트인 선택
4. 단내림 활성화 → 위치: 오른쪽
5. 가구 라이브러리에서 가구 선택
6. 메인 영역과 단내림 영역에 가구 배치 시도
✅ 예상 결과:
  • 가구가 정상적으로 배치됨
  • 콘솔에 "Zone info is null" 에러가 없음
  • 메인 영역과 단내림 영역 모두 가구 배치 가능

테스트 2: 노서라운드 + 세미스탠딩 + 단내림(왼쪽)

1. 새 디자인 생성
2. 서라운드 타입: 노서라운드 선택
3. 설치 타입: 세미스탠딩 선택
4. 단내림 활성화 → 위치: 왼쪽
5. 가구 배치 시도
✅ 예상 결과:
  • 엔드패널이 있는 쪽은 18mm 공간 차감
  • 가구 배치 정상 작동

테스트 3: 노서라운드 + 프리스탠딩 + 단내림

1. 새 디자인 생성
2. 서라운드 타입: 노서라운드 선택
3. 설치 타입: 프리스탠딩 선택
4. 단내림 활성화
5. 가구 배치 시도
✅ 예상 결과:
  • 양쪽 엔드패널 각 18mm 공간 차감
  • 가구 배치 정상 작동

🔍 검증 항목

  • 브라우저 콘솔에서 Zone info is null 에러가 발생하지 않는지 확인
  • 가구를 드래그하여 배치할 때 정상적으로 스냅되는지 확인
  • 메인 영역과 단내림 영역 경계에서 가구 이동이 자연스러운지 확인
  • 각 설치 타입별로 내부 너비가 올바르게 계산되는지 확인

🐛 디버깅 로그 확인

콘솔에서 확인할 로그:
  • 🔍 calculateZoneSlotInfo 시작점 계산 - 시작점과 내부 너비 확인
  • 🎯 [calculateZoneSlotInfo] 최종 계산 결과 - 메인/단내림 구간 정보
  • 🔍 노서라운드 왼쪽/오른쪽 단내림 경계 계산 - 경계 갭 확인
